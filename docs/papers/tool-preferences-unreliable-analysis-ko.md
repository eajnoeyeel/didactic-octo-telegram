# Tool Preferences in Agentic LLMs are Unreliable

> 출처: EMNLP 2025
> 한 줄 요약: 도구 설명의 작은 편집만으로도 LLM의 도구 선택이 크게 변화하며, 설명은 중립 정보가 아닌 "설득 입력(persuasive input)"으로 작동함을 실증한 연구.

---

## 해결하려는 문제

LLM 기반 에이전트가 도구를 선택할 때, 도구 설명(description)의 미세한 변화가 선택 결과에 어떤 영향을 미치는지 분석. LLM의 도구 선호(tool preference)가 설명의 표현 방식에 얼마나 민감하고 불안정한지를 검증하고자 함.

## 핵심 아이디어

도구 설명은 "중립적인 정보(neutral information)"가 아니라 **"설득 입력(persuasive input)"**으로 작동한다는 것. 동일한 기능을 가진 도구라도 설명의 표현 방식에 따라 LLM의 선택이 크게 달라질 수 있으며, 이는 도구 설명의 품질이 선택률에 직접적으로 영향을 미친다는 것을 의미함.

## 방법론

설명의 작은 편집(minor edits) -- 예를 들어 단어 선택, 문장 구조, 어조 변화 등 -- 이 LLM의 도구 선택 결정에 미치는 영향을 체계적으로 측정. 동일 기능의 도구에 대해 다양한 설명 변형을 생성하고, 각 변형에 대한 LLM의 선택 패턴을 분석.

## 주요 결과

- 설명의 **작은 편집**만으로도 선택되는 도구가 **크게 변화**함
- LLM의 도구 선호는 설명 표현 방식에 **불안정(unreliable)**하게 반응함
- 설명은 기능 명세가 아닌 **설득적 텍스트**로 기능함

## 장점

- **핵심 가설 실증**: "설명 품질이 선택률에 영향을 준다"는 직관적 가설을 실험적으로 증명
- **EMNLP 2025 발표**: 최상위 NLP 학회에서 peer-review를 거친 연구
- **실용적 함의**: 도구 제공자가 설명을 "최적화"할 수 있다는 가능성과 그에 따른 공정성 문제 제기

## 프로젝트 시사점

MCP Discovery Platform의 핵심 테제인 **"설명 품질 -> 선택률"의 타당성을 직접 증명**하는 논문. 설명이 "설득 입력"이라는 발견은, 우리 플랫폼에서 DQS(Description Quality Score)를 통해 설명 품질을 측정하고 개선하는 것이 실제로 도구 선택률 향상에 기여할 것임을 강력히 뒷받침함. 동시에, 설명의 "조작 가능성"에 대한 공정성 이슈도 고려해야 함을 시사.

## 적용 포인트

- **프로젝트 핵심 가설 근거**: "설명 품질 -> 선택률" 인과관계의 학술적 근거로 직접 인용 가능
- **DQS 시스템 정당화**: 설명이 persuasive input이므로, 품질 점수를 통해 "좋은 설득"과 "나쁜 설득"을 구분하는 시스템이 필요
- **공정성 가이드라인**: 설명 최적화가 과도한 "마케팅"이 되지 않도록 하는 기준 설계 필요
- **Evidence Triangulation 설계**: DQS와 선택률 간 Spearman 상관(8b)이 유의미하게 나올 것이라는 기대의 근거
- **비즈니스 케이스**: 설명 품질 개선 서비스의 가치를 설명할 때 핵심 레퍼런스
