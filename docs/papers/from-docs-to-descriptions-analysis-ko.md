# From Docs to Descriptions

> 출처: Wang et al., 2025 (arXiv:2602.18914)
> 한 줄 요약: 10,831개 MCP 서버를 초대규모 분석하여 설명 품질 문제를 4개 차원 18개 카테고리로 분류하고, 표준 준수 설명의 도구 선택률이 비준수 대비 260% 높음을 실증한 연구.

---

## 해결하려는 문제

MCP 도구 설명의 품질 문제를 체계적으로 분류(taxonomy)하고, 각 문제 유형이 LLM의 도구 선택 행동에 미치는 영향을 정량적으로 측정하고자 함. 특히 어떤 종류의 설명 결함이 선택률에 가장 큰 영향을 미치는지 규명.

## 핵심 아이디어

MCP 도구 설명의 품질 문제를 4개 상위 차원과 18개 세부 카테고리로 분류하는 taxonomy를 제시하고, 표준 준수(compliant) 설명과 비준수(non-compliant) 설명 간의 도구 선택률 차이를 측정. Functionality 차원이 선택률에 가장 큰 영향을 미침을 발견.

## 방법론

**평가 대상**: 10,831개 MCP 서버 (초대규모)

**문제 카테고리 (4개 차원, 18개 유형)**:
- **Accuracy (7가지)**: 동작 불일치, 경계 모호, 파라미터 의미 오류
- **Functionality (4가지)**: 중복 도구명, 혼동스러운 설명, 트리거 조건 부재
- **Information Completeness (4가지)**: 파라미터 설명 없음, 반환값 문서 부재
- **Conciseness (3가지)**: 중복 정보, 과도한 전문용어, 관계없는 잡음

**검증**: 표준 준수 설명 vs 비준수 설명의 도구 선택률 비교.

## 주요 결과

| 지표 | 값 |
|------|-----|
| 분석 대상 | **10,831개** MCP 서버 |
| 표준 준수 설명의 선택률 | **72%** |
| 비준수 설명의 선택률 | **20%** |
| 선택률 증가 비율 | **260%** (72% / 20%) |
| Functionality 차원의 선택률 영향 | **+11.6%** (가장 큰 영향) |
| Accuracy 차원의 선택률 영향 | **+8.8%** |

## 장점

- **초대규모 데이터셋**: 10,831개 MCP 서버로 MCP 생태계의 광범위한 실태를 파악
- **세분화된 taxonomy**: 4개 차원, 18개 카테고리로 문제를 구체적으로 분류하여 진단 가능
- **차원별 영향력 분석**: 어떤 종류의 문제가 선택률에 가장 큰 영향을 미치는지 정량화
- **실용적 가이드라인 제공**: 문제 유형별 개선 방향을 제시

## 프로젝트 시사점

MCP Discovery Platform의 DQS 설계에 방향성을 제공하는 핵심 논문. 표준 준수 vs 비준수의 260% 선택률 차이는 설명 품질 점수 시스템의 비즈니스 가치를 정량적으로 입증. 논문의 4개 차원(Accuracy, Functionality, Information Completeness, Conciseness)은 DQS 설계에 참고되었으나, 논문에 "Specificity"라는 차원은 존재하지 않음. 프로젝트 DQS의 Specificity 차원은 논문의 Functionality/Accuracy 차원에서 영감을 받은 프로젝트 자체 설계임.

## 적용 포인트

- **DQS 차원 설계 참고**: 논문의 Functionality(+11.6%)와 Accuracy(+8.8%) 차원의 영향력이 프로젝트 DQS 설계에 방향성 제공. 다만 DQS의 Specificity 차원과 가중치 0.25는 프로젝트 자체 설계이며 논문이 직접 제시한 것이 아님
- **문제 taxonomy 활용**: 18가지 문제 카테고리를 Provider Analytics의 구체적 피드백 항목으로 활용 가능
- **선택률 비교 데이터**: 72% vs 20% 수치를 프로젝트의 비즈니스 케이스로 활용
- **Conciseness 차원**: 과도한 정보도 문제가 됨을 확인 -- 설명의 "적정 길이" 가이드라인 근거
- **Evidence Triangulation(8b, 8c)**: DQS와 실제 선택률 간 상관 분석의 기대치 설정에 활용
